Introduction to HTML

HTML stands for HyperText Markup Language. It is used to create the structure of web pages by organizing content into headings, paragraphs, images, and more. HTML is the foundation of all web pages and is interpreted by browsers.

Key Features of HTML

  • Headings
  • Paragraphs
  • Images
  • Links
  • Forms
  • Tables

Basic HTML Example

        <!DOCTYPE html>
        <html>
            <head>
                <title>Example</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
            </body>
        </html>
